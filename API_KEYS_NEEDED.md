# Необходимые API ключи

Приложение запущено и работает! Но для полной функциональности вам нужно добавить следующие API ключи в файл `.env.local`:

## 1. Yandex Vision API (ОБЯЗАТЕЛЬНО для OCR)

**Для чего:** Распознавание текста из изображений банковских выписок

**Как получить:**
1. Зарегистрируйтесь на https://cloud.yandex.ru/
2. Создайте облако и каталог
3. Перейдите в раздел "Сервисные аккаунты"
4. Создайте новый сервисный аккаунт
5. Выдайте ему роль "ai.vision.user" или "editor"
6. Создайте API ключ для этого сервисного аккаунта
7. Скопируйте ключ

**Добавьте в `.env.local`:**
```env
YANDEX_VISION_API_KEY="ваш_ключ_здесь"
```

## 2. OpenAI API (ОПЦИОНАЛЬНО для AI категоризации)

**Для чего:** Улучшенная автоматическая категоризация транзакций с помощью AI

**Как получить:**
1. Зарегистрируйтесь на https://platform.openai.com/
2. Перейдите в раздел "API Keys"
3. Создайте новый ключ
4. Скопируйте ключ

**Добавьте в `.env.local`:**
```env
OPENAI_API_KEY="ваш_ключ_здесь"
```

**Примечание:** Если не добавите OpenAI ключ, приложение будет использовать локальную категоризацию по ключевым словам, что тоже работает, но менее точно.

## 3. NEXTAUTH_SECRET (УЖЕ СГЕНЕРИРОВАН)

Этот ключ уже автоматически сгенерирован и добавлен в `.env.local`. Менять его не нужно.

## После добавления ключей

Перезапустите приложение:
```bash
# Остановите текущий процесс (Ctrl+C) и запустите снова:
npm run dev
```
