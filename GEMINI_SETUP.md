# Настройка Gemini API

Приложение теперь использует Google Gemini API вместо OpenAI для:
- Более точного парсинга банковских выписок
- Улучшенной категоризации транзакций

## Как получить Gemini API ключ

1. Перейдите на https://aistudio.google.com/app/apikey
2. Войдите в свой Google аккаунт
3. Нажмите "Create API Key"
4. Выберите проект или создайте новый
5. Скопируйте созданный API ключ

## Добавление ключа в приложение

Откройте файл `.env.local` и добавьте:

```env
GEMINI_API_KEY="ваш_ключ_здесь"
```

## Преимущества Gemini API

- ✅ Бесплатный тариф (до 60 запросов в минуту)
- ✅ Более точный парсинг банковских выписок
- ✅ Улучшенная категоризация транзакций
- ✅ Поддержка русского языка

## После добавления ключа

Перезапустите приложение:
```bash
# Остановите текущий процесс (Ctrl+C) и запустите снова:
npm run dev
```

Приложение автоматически будет использовать Gemini для:
- Парсинга выписок (более точное извлечение транзакций)
- Категоризации транзакций (более умная категоризация)

Если ключ не указан, приложение будет использовать:
- Обычные парсеры (regex-based)
- Локальную категоризацию по ключевым словам
